# 윈도우 무설치 주식 모니터 (Stock Monitor PWA) 구현 완료 리포트

사용자님의 피드백을 모두 반영하여 **안티바이러스에 안전한 PWA(설치형 웹 앱) 기반의 주식 모니터 프로그램** 개발을 성공적으로 완료했습니다! 🎉

## 🛠️ 개발 완료된 기능
- **PWA 완벽 적용**: 브라우저에서 독립된 윈도우 창으로 열리며, 작업표시줄에 윈도우 기본 프로그램처럼 깔끔하게 고정됩니다.
- **다크 모드 & 위젯 UI**: 한쪽에 작게 띄워두기 좋은 심플한 화면 디자인을 적용했습니다. (Tailwind CSS 활용)
- **네이버 증권 완벽 연동 (CORS 우회 API)**: 서버(Next.js API Route)를 통해 네이버 실시간 주식 데이터를 파싱해 옵니다. 종목의 6자리 코드(예: 삼성전자 005930)를 입력하여 빠르고 정확하게 종목을 조회할 수 있습니다.
- **다중 목표가 및 자동 돌파 판별**: 현재가를 기준으로 입력한 금액이 더 높으면 '상승 돌파', 낮으면 '하락 돌파'로 자동 판별하며 한 종목에 여러 개의 알림을 달 수 있습니다.
- **백그라운드 폴링 & 윈도우 기본 알림**: 10초마다 가격을 확인하다가 목표가에 도달하면 `띠링-` 소리(자체 오디오 신시사이저)와 함께 윈도우 기본 토스트 알림을 띄웁니다.

## 🚀 사용 및 배포 안내

이 프로그램은 "무설치 웹 앱"이므로, 사용자님이 인터넷만 되면 어디서든 접속할 수 있게 **Vercel(무료)**에 배포하는 것이 가장 좋습니다.

### 1단계: Vercel에 무료 배포하기
1. 현재 코드를 사용자님의 GitHub 저장소(Repository)에 업로드(Push)합니다.
2. [Vercel](https://vercel.com/) 사이트에 가입/로그인합니다.
3. `Add New Project`를 누르고, 방금 올린 GitHub 저장소를 선택해 `Deploy` 버튼만 누르면 1분 만에 인터넷 주소가 생성됩니다.

### 2단계: 윈도우 앱으로 설치하기 (무설치)
1. 크롬(Chrome) 브라우저나 엣지(Edge) 브라우저로 배포된 Vercel 주소에 접속합니다.
2. 주소창 맨 우측에 있는 **'앱 설치' 아이콘(또는 메뉴 > [주식 모니터] 설치)**을 클릭합니다.
3. 윈도우 바탕화면과 작업표시줄에 바로가기 아이콘이 생기고, 단독 프로그램처럼 창이 뜹니다.

> [!TIP]
> 이제 창을 작게 줄여서 화면 구석에 두거나, 작업표시줄에 내려둔(최소화) 상태로 두시면 됩니다. 알림이 제대로 오려면 우측 상단 'X' 버튼을 눌러 **완전히 끄지만 않으시면 됩니다.**

### (선택) 즉시 로컬에서 테스트해보기
배포 전에 지금 내 컴퓨터에서 바로 확인해 보시려면 아래 명령어를 실행하세요:
```bash
cd c:/workspace/stock-monitor
npm run dev
```
이후 웹 브라우저에서 `http://localhost:3000` 으로 접속하시면 확인 가능합니다. (로컬 호스트 환경에서도 크롬의 '앱 설치'를 테스트해 볼 수 있습니다.)
